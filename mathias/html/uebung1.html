Aufgabe 01 (Wiederholung HTML und CSS)

  • 1.1 Frischen Sie Ihre HTML- und CSS-Kenntnisse auf, indem Sie den Abschnitt „HTML Basic“ auf: //www.w3schools.com/html/ sowie die Abschnitte „CSS Basic“, „CSS Box Model“ und „CSS Advanced“ auf http://www.w3schools.com/css/ durchgehen.

  • 1.2 Gestalten Sie mit CSS eine moderne Webseite ohne Frames und Tabellen, wobei eine Startseite, wie unten gezeigt, genügt. Allerdings dürfen die wichtigsten Elemente wie Navigationsmenü, Inhaltsbereich, Kopfbereich und Fußzeile nicht fehlen. Zudem darf sich das Layout nicht ändern, wenn man das Browserfenster größer oder kleiner macht und der Inhaltsbereich muss beliebig groß werden dürfen.

Aufgabe 02 (Wiederholung JavaScript)

  • 2.1 Frischen Sie Ihre JavaScript-Kenntnisse auf, indem Sie die Abschnitte „JS Basic“, „JS Advanced“ (ohne RegExp) und „JS HTML DOM“ auf ://www.w3schools.com/js/ durchgehen. Eine deutschsprachige Alternative ist das inzwischen etwas in die Jahre gekommene SELFHTML (://de.selfhtml.org).

  • 2.2 Erstellen Sie ein JSON-File, das für alle Ihre Kursmitglieder Daten wie Vorname, Name und Email-Adresse enthält. Fügen Sie dem Content-Bereich noch einen Button „Lade Daten“ hinzu. Klickt man selbigen, soll der Button verschwinden, die Datei über XHR eingelesen werden und deren Inhalt in ein gültiges JavaScript-Objekt verwandelt werden. Die eingelesenen Daten sollen dann in einer Tabelle, ähnlich wie unten dargestellt, angezeigt werden. Mit den beiden Buttons in der letzten Spalte soll es möglich sein, Personen für eine Arbeitsgruppe dynamisch aus- bzw. einzutragen, welche sodann in einer zusätzlichen Liste darunter angezeigt werden.
